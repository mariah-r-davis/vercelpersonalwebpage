HTML Tables:

HTML Syntax -
***
<table>
    <thead>    
        </tr>
        <th>Header 1</th>
        <th>Header 2</th>
        <th>Header 3</th>
        <tr>
    </thead>
    <tr>
        <td> Cell 1 </td>
        <td> Cell 2 </td>
        <td> Cell 3 </td>
    </tr>
    <tr>
        <td> Cell 1 </td>
        <td> Cell 2 </td>
        <td> Cell 3 </td>
    </tr>
    <tr>
        <td> Cell 1 </td>
        <td> Cell 2 </td>
        <td> Cell 3 </td>
    </tr>
</table>


<table></table> - Creates the table
<tr></tr> - Table Row
<td></td> - Table Cell
<thead></thead> - Table Header
<th></th> - Table Header Cells

***

CSS Syntax -
***
table {
    width: 100%;
}

td {
    border: 1px solid #333;
    padding: 15px;
}

td - Styles the cells
table - Styles the table
